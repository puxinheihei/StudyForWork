# 测试开发秋招学习路线（头部大厂向，重构版）

> 目标：从 2025-12-13 到 2026-03-01，在每天 2–3 小时的前提下，系统夯实算法、Java 后端、Python 自动化、计算机基础与测试开发工程能力，为头部大厂测试开发暑期实习面试做准备。

---

## 一、使用说明与总体原则

- 这份文档只包含：
  - 每周概览：本周要重点完成什么；
  - 每日计划：当天具体学什么、练什么。
- 默认每天可支配学习时间为 2–3 小时，推荐结构：
  - 40–60 分钟：算法（建议 1 道新题 + 1 道复盘题，共 2 题）；
  - 60–90 分钟：Java / Python+实习 / 计算机基础 / 项目实践。
- 可以根据当天实际工作强度微调：
  - 实习任务很重 → 保底完成算法 + 与实习密切相关的部分；
  - 状态较好 → 适当多花一些时间在 Java 基础或难点知识上。
- 算法重复策略：
  - 工作日：主攻本周主题，每天至少复盘 1 道旧题；
  - 周末：优先做混合题（穿插数组/链表/树/图/DP 等）+ 本周错题复盘。

基础覆盖目标：

- 算法：数组、链表、栈队列、哈希、二分、双指针、树、图、回溯、动态规划。
- Java：语法、集合、异常、IO、并发（线程、线程池、锁、可见性）、JVM 基础。
- 数据库：MySQL 基础 CRUD、索引、事务与隔离级别、简单优化思路。
- 基础：计算机网络（TCP/HTTP）、操作系统（进程线程、上下文切换）。
- 测试开发：测试类型、测试设计方法、自动化测试、CI/CD（Jenkins）、缺陷管理（JIRA）。
- 工程实践：一个 Java 练手小项目 + 当前车载音频自动化实习项目。
